    Horizon product pages

    • Horizon UI Plugin — rich UMG dialogue text, decorators, flipbook widgets, and UI component extensions.
    • Horizon Framework Plugin — UMG scene / UI management framework.
    • Horizon Tween Plugin — tween animation system for actors, scene components, and UMG widgets.
    • Horizon Dialogue Plugin — dialogue framework built on the Horizon plugin family.
    • Horizon Quest Plugin — quest workflow and related gameplay systems.
    • Horizon Interact Plugin — interaction workflow and related gameplay systems.

    Technical Details

    List of Modules: HorizonUI (Runtime) Intended Platform: All Platforms Platforms Tested: Win64, Android Widget Introduction: 1. HorizonFlipbookWidget: This widget support using PaperFlipbook directly in UMG! 2. HorizonDialogueMsgTextBlock: Combine both rich text and dialogue text function in this widget! For rich text feature: you can control text Color, Shadow Color, Font, Font size and padding for each text block. For dialogue text feature: you can control dialogue speed and whether it will repeated after finish.

NEW: 
implement HorizonDialogueTextOverflowWarpMethod for UHorizonDialogueMsgTextBlock. You will be able to select "Normal" or "BreakAll" method, by default, plugin will change default warp method from BreakAll to Normal. 

Normal Warp method means if the word overflow current line width(for both CJK or non-CJK), it will try move to next line and use BreakAll rule in next line. BreakAll method is plugin's previously implementation, the word will break at any character when text overflow occur.
